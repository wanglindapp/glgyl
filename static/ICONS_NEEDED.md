# 需要添加的图标文件

## 底部导航栏图标 (必需)

请将以下图标文件添加到 `static/` 目录：

1. **tab-home.png** - 首页图标（未选中状态）
2. **tab-home-active.png** - 首页图标（选中状态）
3. **tab-mine.png** - 我的图标（未选中状态）
4. **tab-mine-active.png** - 我的图标（选中状态）

图标尺寸建议：81px × 81px

## uni-icons 图标说明

项目中使用的 uni-app 内置图标（无需额外添加）：
- person, locked, eye, eye-slash, chatbubble
- calendar, loop, paperplane, checkbox
- location, person-add, wallet, gift, help, gear
- sound, scan, weixin, creditcard, camera
- inbox, list, trending-up, trending-down, cash, minus, add

这些图标都是 uni-app 内置的，应该能正常显示。

## 临时解决方案

如果暂时没有图标文件，可以：
1. 使用纯色块代替
2. 使用文字代替图标
3. 从 iconfont 或其他图标库下载合适的图标
